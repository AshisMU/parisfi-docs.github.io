Tutorial 4: Template Fill (AcroForm PDF)

Learn how to fill standard PDF forms (AcroForms) using data extracted by Parsifi.

1. Upload an AcroForm PDF

Navigate to Templates in Parsifi.

Click Upload Template and select a PDF file that contains interactive form fields (AcroForm).

If you don't have one, search online for "sample AcroForm PDF". A common example is the US IRS W-9 form.

2. Inspect Form Fields

Once uploaded, click the Inspect Required Fields button (eye icon) next to your PDF template.

Parsifi will extract and list every named field in the PDF. Unlike DOCX templates which use `{{placeholders}}`, PDF templates use the underlying names of the form fields themselves (e.g. `topmostSubform[0].Page1[0].f1_01_0_[0]`).

3. Map to JSON Schema

To fill this form automatically in a pipeline, your Agent Tool's output JSON Schema keys must exactly match the PDF form field names.

Navigate to Advanced > JSON Schemas and create a schema where the property names match the physical PDF field names.

If the PDF field name contains special characters (like brackets `[]`), you must use the exact string in your schema definition.

4. Checkboxes and Radio Buttons

To tick a checkbox or select a radio button in a PDF, your LLM must output the specific export value defined in the PDF for that state.

Commonly, this is `"Yes"`, `"On"`, or `"1"`. If you are unsure, you may need to open the PDF in Adobe Acrobat and check the properties of the physical field to see its exact export value when checked.

Verify Success: When a pipeline executes that links your Extractor tool to this PDF Template, a new PDF will be generated with the form fields permanently flattened and filled with the extracted data.
