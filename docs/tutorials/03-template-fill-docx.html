Tutorial 3: Template Fill (DOCX)

Learn how to map extracted JSON data into a Microsoft Word document using Jinja2 curly brace placeholders.

1. Prepare the Word Document

Create a simple Word document on your computer.

Type: "This invoice is for customer {{customer_name}} and the total is ${{total_amount}}."

Save the file as invoice_summary.docx.

2. Upload the Template

Navigate to Templates in Parsifi.

Click Upload Template and select your invoice_summary.docx file.

Once uploaded, click the Inspect Required Fields button (eye icon) next to it. Parsifi will scan the document and list customer_name and total_amount as required variables automatically.

3. Run a Test Fill

On the Templates page, click the Test Fill button (play icon) next to your template.

A modal will appear asking for the values. Enter "Acme Corp" for customer_name and "500" for total_amount.

Click Generate Document. A new completed `.docx` file will be generated and downloaded to your browser.

Verify Success: Open the downloaded Word document. The curly brace placeholders will be completely gone, replaced perfectly by the text you entered, retaining all original Word formatting.
