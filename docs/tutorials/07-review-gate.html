Tutorial 7: Human-in-the-Loop Review Gates

Learn how to pause an Orchestration Pipeline mid-flight, allowing a human user to verify and correct AI-extracted data before continuing.

1

Create a Simple Pipeline

Navigate to Orchestration and click Create Pipeline.

Use the Smart Wizard: define 1 input document (e.g. "Invoice"), select your "Invoice Extractor" tool, and toggle Require Human Review to ON.

Do not select a Generator template yet. Click Create.

2

Inspect the Canvas

You will be taken to the Data Flow Graph.

Notice the topology: there is an Extractor node wired into a yellow Review Gate node.

3

Trigger the Pipeline

Click the Run Pipeline button at the top of the canvas.

Upload a test invoice document for the requested input slot and launch.

The Extractor node will spin indicating extraction is in progress. Once finished, the pipeline status will change to Pending Review.

4

Complete the Human Review

A banner will appear at the top of the screen indicating action is required. Click the Review Needed button.

A modal appears showing the AI's extracted JSON data. Edit one of the values (e.g. overwrite the extracted `total_amount` with a different number).

Click Approve & Continue.

Verify Success: The pipeline status changes to Complete. If you click on the Extractor node, you will see the original data the AI extracted. If you click on the Review Gate node, you will see the modified data that you approved, proving the human intervention overrode the AI.
