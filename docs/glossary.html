Glossary of Terms

A quick reference guide for Parsifi concepts and architecture terminology.

Agent Tool
A persistent, modular configuration built in the Tool Factory. Combines a prompt, an optional output JSON schema, and execution rules for reusable extraction tasks.
Branch Step
A pipeline step in a Workspace that resolves a Jinja2 condition expression to dynamically split logic into YES/NO routing tracks.
Docling
The highly-optimized open-source core IBM engine that Parsifi wraps to handle layout-dense PDF document reading and Markdown conversion.
Data Lineage
The visual traceability of data within a Pipeline's Data Flow Graph, allowing users to see exactly which document and tool produced a specific extracted variable.
Data Lineage Map
An interactive visual overlay on the Pipeline Canvas (toggled via the 👁 Data Map button) that renders variable nodes between Extractor and Generator nodes. Variable nodes are colour-coded: teal = correctly mapped, red = unmet (template requires a variable no Extractor produces), orange = collision (two Extractors produce the same variable name).
Human Review Gate
A node in an Orchestration Pipeline that pauses execution until a human user manually verifies, edits, and approves the data extracted by preceding steps.
Input Slot
An explicit requirement defined in an Orchestration Pipeline (such as a 'Primary Contract' document or 'Company Context' text) that must be fulfilled before the pipeline can run.
Map-Reduce Extraction
The process of sequentially distributing large context windows into discrete, hierarchical chunks so the LLM does not hallucinate, before dynamically merging them to satisfy an overarching query.
Pipeline (Orchestration)
A multi-input, multi-output Directed Acyclic Graph (DAG) that wires Agent Tools to Templates using the Variable Register. Used for complex document generation workflows, unlike the linear Workspace.
Pipeline Version
An immutable, numbered snapshot of a Pipeline's graph (nodes, edges, and configuration) automatically created every time the canvas is saved. Accessible via the 🕐 History button; any past version can be restored to the canvas and re-committed as a new version, preserving the full history.
Recipe
A pre-built, templated Orchestration Pipeline or Workspace that can be cloned and customised to accelerate setup for common industry use cases (e.g. KYC Onboarding).
Reference / Knowledge Base
A dedicated sector within Parsifi where reusable network assets—like previous contracts, AcroForm Templates, or internal procedure docs—reside.
Repeat (Loop) Step
A powerful Workspace node that maps an inner Agent Tool to continuously evaluate and process against an exported list of objects output from a previous step.
Strict Mode
A specific constraint layer appended to a task prompt guaranteeing the AI assumes an exact verbatim, "no-hallucination, no-extrapolation" posture.
Variable Register
The central nervous system of an Orchestration Pipeline. It automatically maps the JSON output keys from Extractor tools to the corresponding `{{placeholders}}` in Generator templates, eliminating the need for manual data mapping.
Workspace
A fully automated, linear business logic pipeline capable of linking Categorisation, extractions, conditional branching, and loops together over a single document. For multi-document routing and template generation, see Pipeline.
